# Optimisation archive

This directory contains historical SmartCopy2026 copy optimisation design notes,
benchmark discovery records, and production validation evidence.

These documents are retained for traceability. They are not the active product
architecture reference; current production behaviour should be documented in
`Docs/Architecture.md`.

The `configs/` directory contains the machine-specific benchmark and validation
inputs used during the investigation. They are retained as evidence and may
refer to local fixtures, drives, and paths; they are not portable run
configurations.
